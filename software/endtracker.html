Software > EndTRACKER

Super-simple web analytics

Key facts

  • Primary language: Rust
  • Creation year: 2021

Homepage

  • /stats.html

EndTRACKER is a super-simple web analytics service custom-built for this site.

This project is a response to my frustration with Google Analytics' complexity and intrusiveness, as well as an experiment to see how much traffic GA doesn’t even notice because of privacy blockers. Other than analytics, this program also powers the post voting features in this site.

This tracker is lightweight (less than 5k SLoC) and privacy-aware: the tracker only uses a cookie to offer page voting functionality and to offer minimal insights on returning visitors, but these cookies are not required and can be rejected. See the privacy page for more details.

EndTRACKER is written in Rust, is deployed as an Azure Functions service, is backed by a PostgreSQL database, and uses Chart.js for graphing. The name of the project resembles EndBASIC because the foundations of the service are directly derived from the code I wrote for the EndBASIC web service.

This project is not free software, mostly because I have no desire right now to run this as an open-source project. You can see how the metrics look like by visiting the statistics page of this blog.

If you have any interest in using EndTRACKER for your site, let’s talk! The backend is designed to support multiple sites so, in principle, I could onboard other people too.
