Software > ticket.el

Ticket/task browser for tk

Key facts

  • Primary language: Elisp
  • Creation year: 2026
  • License: MIT

Homepage

  • https://github.com/jmmv/ticket.el

An Emacs package that provides integration with the ticket (tk) command-line ticket tracking tool. It offers a transient command menu for common ticket operations and an interactive tree-based browser for navigating tickets and their dependencies.

This project has been fully vibecoded as an experiment on how this style of “software development” (these are scary quotes) looks like.
