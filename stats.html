Powered by EndTRACKER

Welcome to super-simple web analytics!

This blog uses EndTRACKER, a home-grown web statistics service that I built specifically for this site. This tracker is a response to my frustration with Google Analytics' complexity and intrusiveness, and EndTRACKER is lightweight (less than 5k SLoC in Rust) and privacy-aware. The name is styled like EndBASIC because the service's foundations are a direct fork of the EndBASIC service. The web UI relies on Chart.js to render the graphs.

I'm showing you the analytics here for transparency and as a demo of what EndTRACKER can do. The project is not open source, but if you are remotely interested in deploying these super-simple statistics to your own site, we can talk!
